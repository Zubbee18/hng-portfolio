# Deborah Anyachukwu — Backend Engineer Portfolio

Portfolio page showcasing my backend work during the HNG internship. Built with React, Vite, and Tailwind CSS.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS v4
- Deployed on Railway

## What This Portfolio Communicates

I wanted a reviewer to open this page, spend 3–5 minutes, and walk away knowing exactly what I can build as a backend engineer. The projects listed are real systems I shipped — not toy demos — and the featured deep-dive shows how I think through architecture decisions.

## Running Locally

```bash
pnpm install
pnpm dev
```

## Building for Production

```bash
pnpm build
pnpm start
```
